A star-filled, privacy-friendly music showcase built with React, Vite, Tailwind CSS, and Framer Motion.

## Getting started

```bash
pnpm install

```

Visit http://localhost:4173 to explore the constellation.

## Build

```bash
pnpm build
```

## Features

- Autoplaying ambient soundtrack with graceful fallbacks.
- Keyboard-friendly, scroll-snap disc slider for curated tracks.
- Lazy player panel that mounts embeds on demand and supports deep-link sharing via `?t=track-id`.
- Toast feedback for ambient autoplay, copy/share events, and blocked audio policies.
- Responsive, accessible design with starry background motion and privacy-respecting embeds.
